# Investigating Resting-State Aftereffects of Cognitive Functions

## About the project

Hi, I’m **Ioanna Prakate**, MSc Neuropsychology graduate.

This project investigates whether different cognitive tasks leave measurable aftereffects in subsequent resting-state fMRI activity and wether BOLD activity accumulates after concecutive tasks.

## Research question

Do lower and higher cognitive functions affect the brain’s resting-state activity differently after task performance?

## Study design

Participants completed:

1. Baseline resting state fMRI  
2. Motor task  
3. Post-motor resting state fMRI  
4. Simon task  
5. Post-Simon resting state fMRI
6. Pinel task
7. Post-Pinel resting state fMRI

## Methods

- Resting-state fMRI
- Task-based fMRI
- Sparse Paradigm Free Mapping
- Comparison of pre-task and post-task resting periods

## Why it matters

Understanding how cognitive effort persists in the brain after a task may help us better understand neural persistence, mental fatigue, and brain recovery mechanisms.
